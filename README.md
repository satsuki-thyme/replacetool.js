# replacetool.js
replace() で置換します。

## 目次
- replacetool.js: ライブラリ本体
- replacetool.min.js: ライブラリ本体（圧縮版）
- sample.html: 使い方のサンプル
- sample.json: JSON のサンプル
- sample.txt: テキストファイルのサンプル

## 説明
- replace() メソッドで置換します
- replace() 内で関数は使えません

## 動作
- 引数で与えた JSON に従って置換操作をします
- 正規表現にマッチさせて置換します
